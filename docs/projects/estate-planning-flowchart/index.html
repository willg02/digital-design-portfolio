Design Process

Problem

The original flowchart was difficult to follow with a non-linear layout that required readers to jump between different areas of the page. Steps were numbered but the visual flow contradicted the logical sequence.

Solution

Redesigned using Mermaid.js for a clean, maintainable diagram that:

  • Flows horizontally in three clear stages
  • Uses color coding to show different types of steps
  • Includes descriptive text for each step
  • Maintains all original information in a more compact format
  • Can be easily updated via text edits (version control friendly)

Technology Choice

Why Mermaid.js?

  • Text-based diagrams that integrate with version control
  • AI can easily help modify and update diagrams
  • Clean, professional rendering
  • WordPress plugin support for easy CMS integration
  • Responsive and accessible output
  • No need for expensive design software subscriptions
